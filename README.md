# booklibrary (with Easy Admin)

Example project that uses:
  - Symfony 5
  - Doctrine (Sqlite DB)
  - Cache (Redis)
  - Encore (frontend with tailwindcss configuration & alpineJS)
  - Easy Admin 3 (backend)
  - Auth
  - User Roles
  - Email
  - Events & Listeners
  - etc.

## how to install

```bash
# run 
git clone https://github.com/n1crack/booklibrary_easyadmin.git <folder_name>

```

```bash
# run 
composer install
```
